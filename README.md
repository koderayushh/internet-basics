# Internet Basics

This repository contains structured, beginner-friendly notes that explain how the Internet and Web work at a fundamental level.
The focus is on understanding core networking concepts, client–server architecture, Internet protocols, and HTTP/HTTPS, which form the foundation of modern web development.

## 📚 What This Repository Covers

### 1️⃣ How the Internet Works ([Notes](./chapters/01-internet-fundamentals.txt))

- What the Internet actually is (infrastructure vs services)
- Difference between Internet and Web
- History of the Web
  - Web 1.0 (Read-only)
  - Web 2.0 (Read & Write)
  - Web 3.0 (Decentralized Web)
- How computers communicate with each other
- How data travels across the world using packets
- Core concepts:
  - Domain Name
  - IP Address (IPv4 & IPv6)
  - MAC Address
  - Routing
- How ISP and DNS work together to deliver data

### 2️⃣ Client–Server Architecture ([Notes](./chapters/02-client-server-architecture.txt))

- Client–Server model explained
- Difference between Client (Browser) and Server (Hosting Computer)
- HTTP request–response cycle
  - Request
  - Processing
  - Response
  - Render
- What happens when you visit a website
- Front-end vs Back-end
- Static vs Dynamic websites
- Web hosting and how it works

### 3️⃣ Internet Protocols ([Notes](./chapters/03-internet-protocols.txt))

- What Internet protocols are and why they matter
- TCP (Transmission Control Protocol)
  - Reliable, connection-oriented communication
  - 3-Way Handshake:
    - SYN
    - SYN–ACK
    - ACK
- UDP (User Datagram Protocol)
  - Fast, connectionless communication
- Difference between TCP and UDP
- Real-world use cases of TCP and UDP

- What HTTP is and how it works
- HTTP versions:
  - HTTP 1.0
  - HTTP 1.1
  - HTTP 2
  - HTTP 3
- HTTP status codes:
  - 1XX – Informational
  - 2XX – Success
  - 3XX – Redirection
  - 4XX – Client Error
  - 5XX – Server Error
- What HTTPS is and why it is more secure than HTTP
- How HTTPS establishes a secure connection
- SSL vs TLS encryption
- Proxy and Reverse Proxy
- How VPN works and why it is used

## 🎯 Purpose of This Repository

The goal of this repository is to build strong conceptual clarity before jumping into frontend or backend development.
Instead of just learning tools and frameworks, this repo focuses on understanding how the web works internally.

## 👤 Who This Is For

- Beginners starting their web development journey
- Learners who want clarity on Internet and networking concepts
- Developers revising web fundamentals and protocols

Feel free to explore the notes, revise concepts, and build on top of this foundation.
